# Habit Tracker Backend – Semana 1

Proyecto correspondiente a la Actividad No. 1 del curso Programación Avanzada.

Este proyecto implementa la configuración inicial del backend para una aplicación de gestión de hábitos utilizando:

- Express.js
- MongoDB Atlas
- Mongoose

---

##  Descripción

La aplicación permite gestionar hábitos mediante un API REST que incluye:

- Crear hábitos
- Listar hábitos
- Eliminar hábitos

Esta entrega corresponde únicamente a la configuración del backend y conexión a base de datos según los requerimientos de la Semana 1.

---

##  Tecnologías utilizadas

- Node.js
- Express Application Generator
- MongoDB Atlas (Base de datos en la nube)
- Mongoose
- dotenv

---

##  Requisitos previos

Antes de ejecutar el proyecto debes tener instalado:

- Node.js (versión 18 o superior recomendada)
- npm
- Cuenta en MongoDB Atlas
